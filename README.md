# South Africa Trip PWA

这是一个隐私优先版本：
- 不内置任何机票/酒店/租车 PDF
- 附件由你在手机端本地导入
- 文件只保存在当前浏览器本地存储，不上传到服务器

## 使用

1. 本地启动：

```bash
python3 -m http.server 8080
```

2. 手机和电脑同一网络，打开：`http://<你的电脑局域网IP>:8080`
3. 在页面中导入 3 份 PDF（机票/租车/酒店）
4. 浏览器中执行“添加到主屏幕”

## 部署注意

如果部署到公网（如 GitHub Pages），代码可公开，但你的 PDF 不会随代码上传。
